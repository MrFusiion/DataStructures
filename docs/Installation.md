# Installation

### Method 1 - Wally

1. Setup [Wally](https://wally.run/) by using `wally init`.
2. Add `howmanysmall/datastructures` as a dependency.

```toml
[dependencies]
DataStructures = "howmanysmall/datastructures@^1.1.0"
```
